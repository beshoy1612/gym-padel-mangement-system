A comprehensive system to manage gym memberships, staff, class scheduling, and Padel court bookings. ## ğŸ“‹ Features

### Gym Member & Staff Management
- **Members:** ID, Name, Date of Birth, Subscription Period (monthly, 3 months, 6 months, yearly) with pricing.
- **Staff Roles:** Receptionists, Coaches, Managers.
- Staff can store, retrieve, and search member information.
- Automatic notifications for subscription renewals.

### Gym Classes Scheduling & Waitlist
- Monthly class scheduling.
- Coaches have class assignments for quick lookup.
- Each class has a capacity limit.
- Waitlist system: if a class is full, members can join a waitlist.
- Automatic notifications when a spot opens up due to cancellations.

### Workout History Tracking
- Members can track past workouts and monitor progress.

### Subscription Management & Renewal Reminders
- Tracks subscription expiry dates.
- Sends automatic renewal reminders.
- Discounts/offers for early renewals.

### Padel Court Booking System
- Search courts by date, time, and location.
- Shows nearest available time slots if requested slot is unavailable.
- Cancellation/rescheduling allowed up to 3 hours before booking.

### Bonus Features
- Generate monthly reports of most active members and revenue.
- VIP members get priority in bookings and waitlists.
- Optional GUI implementation.
